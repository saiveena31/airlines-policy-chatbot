# airlines-policy-chatbot
✈️ Airlines HR Policy Q&amp;A Bot  This is a Streamlit app that answers questions about **Flykite Airlines HR Policies** using Retrieval-Augmented Generation (RAG).  ## Features - PDF stored on Hugging Face Hub - Fetches PDF dynamically at runtime - Uses FAISS + HuggingFace embeddings for retrieval - Powered by Groq LLM for fast responses
